# Changelog

## [Версия 2.0.0] - 2026-05-17

### 🔐 Исправления безопасности
- **КРИТИЧНО**: Добавлено хеширование паролей с werkzeug.security

- **ВЫСОКАЯ**: Добавлена CSRF защита через Flask-WTF

### ⚠️ Breaking Changes
- Пароли старых пользователей больше не будут работать (нужна переинициализация)
- Требуется Flask-WTF для всех форм (CSRF токены обязательны)

### 📝 Миграция с предыдущей версии
1. Обновите requirements.txt: `pip install -r requirements.txt`
2. Все пользователи должны изменить пароли через "Забыли пароль"
3. Убедитесь, что все формы содержат CSRF токены
4. Запустите тесты: `pytest tests/`
